Development of CPS in Real-Time Bolt Defect Detection

Machine Learning Based Automated Quality Control System

📌 Project Overview

This project focuses on developing a Cyber-Physical System (CPS) for real-time bolt defect detection using machine learning and computer vision. The system automates quality inspection in industrial environments by identifying defects such as cracks, rust, deformation, and missing bolts.

🎯 Objectives

  • Automate industrial quality inspection using ML
  • Detect bolt defects in real-time
  • Reduce human error and inspection time
  • Improve manufacturing efficiency

⚙️ System Architecture

  • Image Acquisition using Camera
  • Preprocessing of Captured Images
  • Object Detection using YOLOv8 Model
  • Defect Classification
  • Output Display / Decision System

🧠 Technologies Used

  • Machine Learning & Deep Learning
  • Computer Vision (OpenCV)
  • YOLOv8 Object Detection
  • Python Programming
  • Embedded Systems (CPS Integration)

🔍 Types of Defects Detected

  • Cracks
  • Rust
  • Deformation
  • Missing Bolts

📊 Features

  • Real-time detection
  • High accuracy using deep learning
  • Automated decision making
  • Scalable for industrial environments

🚀 Applications

  • Manufacturing industries
  • Automotive assembly lines
  • Quality control systems
  • Smart factories (Industry 4.0)

⚠️ Challenges

  • Lighting variations in industrial environments
  • Different bolt shapes and sizes
  • Dataset limitations
  • Real-time processing constraints

💡 Future Improvements

  • Expand dataset for better accuracy
  • Edge deployment using embedded devices
  • Integration with IoT systems
  • Advanced defect classification
